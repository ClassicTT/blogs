GTA San Andreas - Interface Remastered Project v0.8

This project's mission is to improve the interface of the game using modifications created by the community.
These mods will work with SAMP and MTA, it is recommended to downgrade your game if you run Steam version.
By installing those mods you are breaking the Rockstar end user license agreement which means their support will not be able to assist you.

DOWNLOAD



HOW TO INSTALL?

Extract archive contents to your game directory and replace original files.
That's it you can now play the game! If you want full manual installation or looking for creators check list below, every mod used is there.
And if you want original HUD positioning simply remove HD HUD inside modloader folder.
